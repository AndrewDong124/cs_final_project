For the project, my idea was to have a game similar to the Binding of Isaac, being a dungeon crawler combined with roguelike style elements. However, I will likely not make the game as complicated as Isaac was, due to time constraints and the fact that the game is extremely big and decently complex. As my plan involves creating a game with time-based elements and constant input from the player, it will use Python and Pygame, as well as random to generate different stages and levels. To make some of the finer art in the game, external drawing software will probably also be used.
